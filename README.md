# E-commerce Web Platform – PW-IPCA

Aplicação web de e-commerce desenvolvida com **React** (frontend) e **Node/Express** (backend). Inclui autenticação de utilizadores, rotas protegidas, navegação por categorias, gestão de produtos, carrinho de compras, helpdesk e comunicação com API.  
Projeto desenvolvido para a unidade curricular **Programação Web** (2.º ano, 2.º semestre) da Licenciatura em Engenharia de Sistemas Informáticos, no **IPCA**.

## Funcionalidades principais

* Autenticação de utilizadores (login, registo, recuperação de password).
* Rotas protegidas com React Router e Context API.
* Gestão de perfil do utilizador.
* Listagem, pesquisa e filtragem de produtos.
* Carrinho de compras com estado persistente.
* Gestão de produtos, campanhas e pedidos de suporte (helpdesk manager).
* Submissão e acompanhamento de pedidos de helpdesk pelos utilizadores.
* Backend em Node.js/Express com persistência de dados em ficheiros JSON.
* Estrutura modular: controllers, routes e camada de dados.
* Interface responsiva desenvolvida com componentes React reutilizáveis.

# E-commerce Web Platform – PW-IPCA

E-commerce web application built with **React** (frontend) and **Node/Express** (backend). Includes user authentication, protected routes, category browsing, product management, shopping cart, helpdesk features, and full API communication.  
Developed for the **Web Programming** course (2nd year, 2nd semester) of the Computer Systems Engineering degree at **IPCA**.

## Main Features

* User authentication (login, registration, password recovery).
* Protected routes using React Router and Context API.
* User profile management.
* Product listing, search and category filtering.
* Shopping cart with persistent state.
* Backoffice features: product, campaign and support ticket management.
* Helpdesk ticket submission and tracking for users.
* Node.js/Express backend with JSON file-based persistence.
* Modular architecture with controllers, routes and data layer.
* Responsive interface built with reusable React components.
